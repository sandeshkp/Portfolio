1) PRODUCTION PLANNING SOFTWARE

1. Project Overview

This project is built in Flask software — lightweight but capable of handling large data volumes. The number of users accessing this software at a time is limited, mostly during specific hours in the morning.

The project focuses on daily production/mould planning, where product norms and requirements are received from dealers based on previous sales. This project simplifies and speeds up the process of planning production or machine setup according to the moulds generated by the software, based on daily norms and requirements.

2. Business Motivation

This project was initiated to solve challenges in production planning, which was previously managed by one or two people in each production unit across the country. They used to receive requirements and norms from their respective states in the form of Excel sheets.

Additionally, they often received frequent calls from ASM/RSMs of each state to stop or change production for high-demand products, creating confusion and frequent machine stoppages.

The main concern was that machines should not stop — they must run 24/7 with a relevant mould running inside. Some machines also face shutdowns due to electrical or mechanical issues, which must be identified. Each machine has specific mould compatibility (for example, PU, Non-PU, etc.).

There are also linear and circular machine plants, which need to be managed separately. The main goal is to ensure relevant products are produced daily, avoiding stockouts or cut cases (non-relevant, unsold, or defective products stored without use).

3. Impact & Outcomes

The primary and most significant benefit achieved through this project is time efficiency.

Previously, production planning required individuals to collect data from RSMs, ASMs, and dealers across states. The plan was prepared manually, involving multiple calls, consuming almost half a day, and always planning one day behind the actual demand.

With this project, production planning has become live and standardized, reducing the planning time by around 90% compared to the previous process.

Machines now operate continuously without unnecessary stoppage. The software helps identify and utilize each machine effectively. Workers can now easily view which product is next, which mould to use, how many rotations to run, and which machines are inactive — making the entire system reliable, cost-effective, and market-relevant.
