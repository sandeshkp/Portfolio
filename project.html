Production Planning Software

Overview

An internal Flask-based production planning system designed to automate daily mould and machine planning across manufacturing units.

Business Problem

Production planning was manually done using Excel sheets and phone calls, causing frequent machine stoppages, delayed planning, and inefficiencies.

Solution

The application converts dealer demand and sales history into optimized machine and mould schedules while respecting machine compatibility and plant constraints.

Impact

• Planning time reduced by ~90%
• Continuous machine utilization
• Reduced stockouts and cut cases
