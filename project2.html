This project is an internal analytics platform developed to centralize reporting across Sales, Production and Supply Chain operations.

The system consolidates operational datasets including dealer sales, stock availability and production metrics and converts them into analytical dashboards accessible across the sales hierarchy.

Key Objective

• Centralize operational reports into one analytics platform

• Provide sales hierarchy visibility from national level to dealer level

• Automate manual Excel based reporting

• Deliver secure role based dashboard access

Solution Approach

• Python ETL pipelines for SAP HANA data extraction

• Multi-year historical data processing

• Power BI analytical dashboards

• Embedded dashboards inside Flask application

Operational Result

• Removed manual reporting workflows

• Reduced reporting delays

• Centralized analytics access

• Improved decision making

The sales department uses multiple reports such as dealer wise sales, new product launch tracking and daily live sales dashboards.

• Previous day sales

• Month to date sales

• Year to date sales

• Target achievement percentage

• Year over year growth

Earlier these reports were generated in Excel and distributed by email, which created operational delays and data management issues.

SAP HANA → Python ETL → MySQL / PostgreSQL → Power BI → Flask Application

This architecture separates heavy data processing from the reporting layer to ensure scalability and performance.

Data is extracted from SAP HANA through OData APIs and processed through automated Python ETL pipelines.

• Data extraction

• Cleaning and transformation

• Chunk processing for large datasets

• Storage in relational databases and cloud storage

The system provides multiple operational dashboards for sales monitoring.

Dealer Sales New Launch Tracking Daily Live Sales MTD Analysis YTD Growth

A Flask web application acts as the delivery interface for dashboards. Power BI dashboards are embedded using iframe integration.

• Row Level Security implemented

• Region specific dashboard access

• Department level visibility

The platform replaced manual Excel reporting with a centralized analytics system.

• Faster reporting

• Reduced operational workload
